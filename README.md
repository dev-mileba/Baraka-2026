# My Notebooks — Baraka 2026

A collection of Jupyter notebooks tracking a Python programming course, organized by month and week.

## Notebooks

| Notebook | Topics Covered |
|---|---|
| `month_1_week_2.ipynb` | Python basics — Hello World, variables, importing modules |
| `month_1_week_3.ipynb` | Data types, variable naming conventions, string formatting |
| `month_2_week_1.ipynb` | Functions — defining, parameters, return statements |
| `month_2_week_2.ipynb` | Hospital Management System project — CRUD operations with pandas |
| `month_2_week_3.ipynb` | Error handling, file I/O, intro to statistics & probability |
| `month_2_week_4.ipynb` | Statistics, probability, and Exploratory Data Analysis (EDA) |

## Dataset

- `Titanic-Dataset.csv` — Used in EDA exercises (Month 2, Week 4)

## Other Files

- `class_note.txt` — Auto-generated by `month_2_week_3.ipynb` as a file handling exercise

## Setup

This project uses a Python virtual environment (`.venv`). To get started:

```bash
# Activate the virtual environment
source .venv/bin/activate

# Launch Jupyter
jupyter notebook
```

### Key Dependencies

- `pandas` — data manipulation
- `numpy` — numerical computing
- `seaborn` / `matplotlib` — data visualization

## Progress

- **Month 1:** Python fundamentals (variables, data types, syntax)
